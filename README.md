# png-to-ico

![GitHub last commit](https://img.shields.io/github/last-commit/KOTO28/png-to-ico)

English | [日本語](README_ja.md)

A simple Python script to convert PNG images to ICO format.

## Features

- Convert PNG images to ICO format
- Support for multiple sizes (16x16, 32x32, 48x48, 64x64, 128x128, 256x256)
- Preserve transparency in PNG images

## Usage

Place/rename your PNG as `input.png` in the project root before running.

```bash
# Install dependencies
pip install -r requirements.txt
# Run the conversion script
python make_ico.py
```

## Notes

- Tested with Python 3.12.9.
