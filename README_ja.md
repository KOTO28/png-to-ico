# png-to-ico

![GitHub last commit](https://img.shields.io/github/last-commit/KOTO28/png-to-ico)

[English](README.md) | 日本語

PNG画像をICO形式に変換するシンプルなPythonスクリプト

## 特徴

- PNG画像をICO形式に変換
- 複数サイズのサポート (16x16, 32x32, 48x48, 64x64, 128x128, 256x256)
- PNG画像の透過性を保持

## 使い方

プロジェクトのルートディレクトリにPNGファイルを `input.png` として配置/リネームしてから実行してください。

```bash
# 依存関係のインストール
pip install -r requirements.txt
# 変換スクリプトの実行
python make_ico.py
```
